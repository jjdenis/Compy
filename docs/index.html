The Compy Project - Index

The Compy Project is an personal project of a father of four, who learnt programming with a Commodore Vic-20 and, much later, felt in love with python.

The Compy Project aims to blend the beautiful, colorful and easy screen familiar to Commodore users with the elegant syntax of python.

To start, check how to intall it, then run the examples, and then, and only then, try the challenges.
