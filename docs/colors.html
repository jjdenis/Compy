Color Codes

These are the meaning of the codes used in commands like sc.printf() or sc.set_bg_color().

  • Code: 0 - Name: light_blue
  • Code: 1 - Name: blue
  • Code: 2 - Name: light_green
  • Code: 3 - Name: green
  • Code: 4 - Name: pink
  • Code: 5 - Name: red
  • Code: 6 - Name: light_orange
  • Code: 7 - Name: orange
  • Code: 8 - Name: light_violet
  • Code: 9 - Name: violet
  • Code: 10 - Name: yellow
  • Code: 11 - Name: brown
  • Code: 12 - Name: white
  • Code: 13 - Name: light_grey
  • Code: 14 - Name: grey
  • Code: 15 - Name: dark_grey
  • Code: 16 - Name: black
  • Code: 17 - Name: cyan
  • Code: 18 - Name: purple
  • Code: 19 - Name: deep_pink
